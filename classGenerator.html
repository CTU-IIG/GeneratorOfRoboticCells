Detailed Description

It encapsulates the RoboticLine and XmlWriter classes and adds the additional features.

Auxiliary class, encapsulating the RoboticLine and XmlWriter classes, helps to generate and write project instances. It adds the features like text progress bar, time measurement, and so on.
